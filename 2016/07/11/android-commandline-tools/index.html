使用命令行管理Android SDK

查看索引

android list sdk --all

安装指定序号

android update sdk -u -a -t 1,2,3,4,..,n
各参数含义通过android update sdk --help查看

项目中的配置是

按照1中的列表序号执行如下操作，序号可能有变化，请安实际情况填写序号
android update sdk -u -a -t 1,2,5,28,139,140
