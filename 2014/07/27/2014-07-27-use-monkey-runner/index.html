MonkeyRunner使用入门

发表于 2014-07-27   |   分类于 Python   |  

monkeyrunner是Android SDK中的一个命令（./sdk根目录/tools），它可以执行事先编写好的Python脚本向手机发送操作指令。

monkeyrunner提供以下三个类来进行手机指令的相关操作：

  • MonkeyDivice

  • MonkeyImage

  • MonkeyRunner

MonkeyDivice

主要负责向手机发送指令

MonkeyImage

可以进行截屏，并比较两幅截图的相似度。

MonkeyRunner

控制本次测试任务
