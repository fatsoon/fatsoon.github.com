新浪微博封杀的API

发表于 2014-08-02   |   分类于 Internet   |  

最近调用新浪微博API时发现一些接口并没有按照文档说明进行返回数据，而是被加了限制，文档中没有做出说明。

  • friendships/friends

    新浪的通知：“为进一步保护用户数据，即日起微博开放平台将对用户关系读取类接口进行升级，各接口最多返回指定用户关注数/粉丝数30%的数据。本次调整涉及所有获取粉丝或粉丝id、关注列表或关注id列表接口。”
